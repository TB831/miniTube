## Description
miniTube is a miniature version of YouTube. It is a application built with React components and utilizes the YouTube API to allow the users to search for videos. Video search is updated dynamically by every keystroke but cannot make more than 1 request every 3 seconds.  

## Getting Started
```
npm install
npm start
```
